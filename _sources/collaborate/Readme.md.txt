Collaborate with PSLab!
=======================

Pocket Science Lab Experiments
------------------------------

The PSLab repository allow contributors from anyone. This is how to
create experiments for [Pocket Science
Lab.](https://github.com/fossasia?q=pslab).

### Instructions

1.  Fork the repo
2.  Go to the [notex](https://www.notex.ch), an
    [open-source](https://github.com/hsk81/notex-v2.0) ReST editor.
